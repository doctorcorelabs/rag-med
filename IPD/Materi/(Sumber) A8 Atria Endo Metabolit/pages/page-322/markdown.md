# Case Description

Ny. Kaio, usia 70 tahun datang ke RS dengan keluhan terasa ada benjolan di leher sejak 3 bulan yang lalu dan dirasakan semakin membesar disertai suara serak dan sesak nafas. Pada pemeriksaan teraba benjolan di colli sinistra ukuran 3 x 4 cm, batas tidak tegas, berbenjol konsistensi keras, tidak bergerak ke atas saat menelan, disertai pembesaran KGB ipsilateral. Pada pemeriksaan histopatologi ditemukan Psamomma bodies.
