# DM Tipe I vs Tipe II

|  Parameter | Tipe 1 | Tipe 2  |
| --- | --- | --- |
|  Usia saat presentasi | Masa kanak-kanak | Pubertas hingga dewasa  |
|  Tipe krisis hiperglikemia | Lebih sering KAD | Lebih sering HHS  |
|  Gangguan insulin | Gangguan sekresi | Gangguan resistansi  |
|  Berat badan | Biasnya kurus | Biasanya obesitas  |
|  Obese/overweight | 20-25 % overweight | >80% obese  |
|  C-peptide | Rendah | Tinggi  |
|  Terapi insulin | Selalu dibutuhkan | Dibutuhkan pada tahap akhir  |
