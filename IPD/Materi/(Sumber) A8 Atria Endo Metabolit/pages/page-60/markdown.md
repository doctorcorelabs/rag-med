# Sindrom Cushing

Definisi: Gangguan endokrin akibat kadar kortisol berlebih di dalam tubuh

Klasifikasi:

|  Diagnosis | Keterangan | Penyakit  |
| --- | --- | --- |
|  Hiperkortisolisme eksogen | Penggunaan kortikosteroid jangka panjang | Sindrom Cushing Eksogen  |
|  Hiperkortisolisme primer | Produksi kortisol berlebihan | Adenoma adrenal (termasuk sindrom Cushing)  |
|  Hiperkortisolisme sekunder | Produksi ACTH berlebihan  |   |
|   |  • Dari hipofisis | Penyakit Cushing  |
|   |  • Dari tumor ektopik penghasil ACTH | Mis. small cell lung cancer (termasuk sindrom Cushing)  |
