# Hipoglikemia

Tata Laksana → Target GDS ≥70 mg/dL

- Hipoglikemia ringan
1. Glukosa oral 2-3 sdm (15-20 g) → tata laksana awal hipoglikemia ringan
2. Monitor GDS setiap 15 menit
3. Bisa diulang hingga 3 siklus
4. Bila tidak berhasil, D10% 150-200 mL dalam 15 menit
