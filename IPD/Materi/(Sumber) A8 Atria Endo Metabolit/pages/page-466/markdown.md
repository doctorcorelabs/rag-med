# Prolaktinoma

## Manifestasi Klinis:

- Gejala hiperprolatinemia
- Pria: libido ↓, impotensi, disfungsi ereksi, oligozoospermia
- Wanita: oligomenore/amenore, infertilitas, galaktore
- Anak: pertumbuhan terhambat, pubertas terlambat, amenore primer

- Gejala lainnya
- Osteopenia
- Gangguan mental: cemas, depresi, mudah lelah, instabilitas emosi
- Gigantisme, akromegali
