# Hipertiroidisme

## Manifestasi Klinis

- Tirotoksikosis → metabolisme meningkat
- Intoleransi panas
- Penurunan berat badan
- Hiperaktivitas simpatis → takikardia, tremor, diaphoresis
- Diare
- Oftalmopati → eksoftalmus, diplopia, retraksi palpebra
- Dermopati → myxedema pretibia akibat penumpukan glikosaminoglikan (bersifat non-pitting) dengan permukaan seperti kulit jeruk
