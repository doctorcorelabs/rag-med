# Hipoparatiroidisme

Definisi: Kondisi dimana terjadi penurunan produksi hormon paratiroid (PTH) dari kelenjar paratiroid

## Etiologi:

- Iatrogenik → Post-operatif → akibat kerusakan kelenjar paratiroid tidak disengaja saat operasi (tiroidektomi, paratiroidektomi, diseksi leher radikal)
- Autoimun
- Kongenital → aplasia atau hipoplasia kel. paratiroid
