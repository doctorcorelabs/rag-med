# Penyakit Addison

## Patofisiologi:

- Kerusakan kelenjar adrenal → ↓ produksi hormon adrenokortikal (aldosterone, kortisol, androgen)

|  Parameter | Tanda dan Gejala  |
| --- | --- |
|  Hipoaldosteronisme | Hiponatremia, Hiperkalemia, Asidosis metabolik  |
|  Hipokortisolisme | Hiperpigmentasi, Hipoglikemia, Hipotensi  |
|  Hipoandrogenisme | Libido menurun, Spermatogenesis menurun  |
