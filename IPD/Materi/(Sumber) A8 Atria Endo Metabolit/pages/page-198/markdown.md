# Goiter Toksik Multinodular

## Tata Laksana:

### Obat antitiroid

- Methimazole → DOC pasien tidak hamil, dapat diberikan pada pasien hamil di trimester 2 dan 3
- Dosis awal 15-30 mg/hari
- Dosis pemeliharaan 5-10 mg/hari
- Propylthiouracil (PTU) → DOC pada kehamilan di trimester 1
- Dosis awal 3x100 mg/hari
- Dosis pemeliharaan 100-200 mg/hari
