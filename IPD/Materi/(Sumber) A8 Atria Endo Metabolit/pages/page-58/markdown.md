# Aksis HPA

Sekresi kortisol melibatkan 3 organ, yaitu:

- Hipotalamus: menghasilkan CRH (Corticotropin releasing hormone)
- Hipofisis anterior: menghasilkan ACTH (Adrenocorticotropin hormone)
- Adrenal: menghasilkan kortisol
