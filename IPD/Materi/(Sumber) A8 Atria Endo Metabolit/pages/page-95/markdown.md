# Pubertas

Tanner Stage Genital (Laki-laki)

|  Tanner Stage | Perkembangan Genital  |
| --- | --- |
|  G1 | Penampilan dan ukuran **prepubertas** dari testis, skrotum, dan penis  |
|  G2 | • Volume testis 4 mL
• Skrotum membesar
• Penis belum membesar
• Perubahan warna dan tekstur skrotum  |
|  G3 | • Testis dan skrotum semakin besar
• Penis mulai membesar  |
|  G4 | • Volume testis 12 mL
• Pertumbuhan penis dan glans penis  |
|  G5 | Penampilan dan ukuran **dewasa** dari testis, skrotum, dan penis  |
