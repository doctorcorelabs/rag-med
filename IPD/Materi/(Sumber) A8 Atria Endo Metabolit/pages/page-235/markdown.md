# Tiroiditis Hashimoto

## Etiologi:

- Idiopatik
- Aktivasi respon imun selular dan humoral → aktivasi limfosit B → produksi antibodi thyroid peroxidase (TPO) dan thyroglobulin (Tg)
- Berhubungan dengan non-Hodgkin limfoma dan penyakit autoimun lain (DM tipe 1, SLE, penyakit Grave, penyakit Addison)
