# Defisiensi Vitamin B

|  Defisiensi | Penyebab | Penyakit  |
| --- | --- | --- |
|  Vitamin B_{7} (biotin) | Malnutrisi, penggunaan antibiotik jangka panjang | Alopesia, dermatitis  |
|  Vitamin B_{9} (asam folat) | Malnutrisi (tea and toast diet), malabsorpsi | Anemia makrositik, spina bifida  |
|  Vitamin B_{12} (kobalamin) | Malnutrisi, Malabsorpsi, reseksi ileum terminalis | Anemia makrositik + neuropati  |
