## Thyroid Storm

- Definisi → emergensi endokrin karena tirotoksikosis berat
- Etiologi → penyakit tiroid tidak terkontrol atau iatrogenic
- Manifestasi klinis → Demam, tirotoksikosis, manifestasi SSP dan GIT
- Pemeriksaan penunjang → tidak diperlukan
- Tata laksana:
- Beta bloker → propranolol
- PTU
- Cholestyramine
- Paracetamol
