# Diabetes Melitus

## Pemeriksaan Penunjang

- Gula darah → hiperglikemia

|  Pemeriksaan | Deskripsi | Nilai  |
| --- | --- | --- |
|  Gula Darah Puasa (GDP) | Pemeriksaan setelah puasa min. 8 jam | Normal: <100 mg/dL  |
|   |   |  Pre-diabetes: 100-125 mg/dL  |
|   |   |  Diabetes: ≥126 mg/dL  |
|  Gula Darah 2 jam PP (GD2PP) | Pemeriksaan setelah konsumsi glukosa 75 gram (TTGO 75 gram) | Normal: <140 mg/dL  |
|   |   |  Prediabetes: 140-199 mg/dL  |
|   |   |  Diabetes: ≥200 mg/dL  |
|  Gula Darah Sewaktu (GDS) | Tidak ada syarat khusus | Normal: <200 mg/dL  |
|   |   |  Diabetes: ≥200 mg/dL  |
