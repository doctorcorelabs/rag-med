# Hipersekresi GH

## Pemeriksaan Penunjang

### Analisa hormon

- Konsentrasi IGF-1 serum
- Meningkat → suspek akromegali/gigantisme, lanjut TTGO
- Normal → rule out akromegali/gigantisme
- Tes toleransi glukosa oral → GH baseline dan setelah 2 jam
- Supresi GH → rule out akromegali/gigantisme
- Tidak ada supresi GH → konfirmasi akromegali/gigantisme
- MRI pituitari
- Konfirmasi adenoma pituitari
