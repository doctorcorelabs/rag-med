# Hipotiroidisme

## Pemeriksaan Penunjang

- TSH dan FT4 → pemeriksaan awal (lihat slide berikutnya)

|  Parameter | Nilai Normal Dewasa  |
| --- | --- |
|  TSH | 2 - 10 mU/L  |
|  fT4 | 0,8 – 2 ng/dL  |
|  Total T3 | 75-195 ng/dL  |

- Antibodi terhadap TPO → bila dicurigai Tiroiditis Hashimoto
- CT-scan / MRI kepala
- Hiperplasia hipofisis pada hipotiroidisme primer
- Tumor / kelainan lainnya pada hipotiroidisme sentral
