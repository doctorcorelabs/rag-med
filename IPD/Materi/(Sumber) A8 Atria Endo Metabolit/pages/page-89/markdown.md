# Pubertas

**Definisi:** Tahap perkembangan antara masa kanak-kanak dengan maturasi fungsional dari kelenjar reproduktif dan genitalia eksternal

## Terminologi:

- Adrenarche → aktivasi produksi hormon androgen adrenal
- Gonadarche → aktivasi kelenjar reproduktif oleh FSH dan LH
- Thelarche → onset perkembangan payudara
- Pubarche → onset tumbuh rambut kemaluan
- Menarche → onset perdarahan menstrual
