# Pubertas

## Fisiologi

- **Perempuan**
- Usia onset: **8-13 tahun**
- Adrenarche → gonadarche → **thelarche** (8-11 tahun) → **growth spurt** (11,5-16,5 tahun) → **pubarche** (rata-rata 12 tahun) → **menarche** (rata-rata 13 tahun)

- **Laki-laki**
- Usia onset: **9-14 tahun**
- Adrenarche → **gonadarche** (9-14 tahun) → **pubarche** (rata-rata 13,5 tahun) → **growth spurt** (rata-rata 13,5 tahun) → **pertumbuhan rambut** ketiak, kumis, dan jenggot
