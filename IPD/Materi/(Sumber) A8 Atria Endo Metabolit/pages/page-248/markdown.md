# Tiroiditis Subakut

## Pemeriksaan Penunjang:

- USG tiroid
- Daerah hipoekoik → Cobblestone appearance
- Temuan histologis
- Tiroiditis subakut granulomatosa → inflamasi granulomatosa, multinucleated giant cell
- Tiroiditis subakut limfositik → Infiltrasi limfositik, titik pusat germinal tipis
