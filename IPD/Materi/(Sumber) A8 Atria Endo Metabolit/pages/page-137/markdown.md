# Defisiensi Vitamin B

|  Defisiensi | Penyebab | Penyakit  |
| --- | --- | --- |
|  Vitamin B_{1} (tiamin) | Alkoholisme, infus tinggi glukosa | Ensefalopati Wernicke, Beri-beri  |
|  Vitamin B_{2} (riboflavin) | Malnutrisi (tidak mengandung laktosa) | Cheilitis, vaskularisasi kornea  |
|  Vitamin B_{3} (niasin) | Defisiensi triptofan | Pellagra  |
|  Vitamin B_{5} (asam pantotenat) | Malnutrisi | insufisiensi adrenal, alopesia  |
|  Vitamin B_{6} (piridoksin) | Alkoholisme, penggunaan isoniazid | Anemia sideroblastik, neuropati perifer  |
