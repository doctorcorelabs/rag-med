# Osteoporosis

## Manifestasi Klinis:

- Umumnya asimtomatis
- Fraktur patologis → fraktur akibat trauma minor atau kegiatan sehari-hari (mis. batuk, membungkuk)
- Fraktur kompresi vertebra → tersering
- Tinggi badan berkurang
- Kifosis → Dowager's hump
- Fraktur lain → collum femoris, radius distal, tulang-tulang panjang
