# Osteoporosis

## Pemeriksaan Penunjang:

- Fungsi ginjal &amp; tiroid, tingkat vitamin D &amp; kalsium darah
- Dual X-ray absorptiometry (DXA) → tes terbaik untuk menilai BMD
- Menggunakan dua sinar X-ray pada vertebra lumbal dan collum femoris
- Hasil dalam T-score:
- &lt; -2,5 SD → Osteoporosis
- -1 hinga -2,5 SD → Osteopenia

T-score adalah perbedaan dalam standar deviasi (SD) BMD pasien dengan rerata BMD perempuan dewasa muda
