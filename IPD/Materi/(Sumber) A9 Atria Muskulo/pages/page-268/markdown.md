# Fraktur Klavikula

Anatomi: Klavikula merupakan tulang berbentuk S dan dibagi menjadi 3 bagian berdasarkan klasifikasi Allman.

- 1/3 tengah (grup I) [69%]
- 1/3 lateral (grup II) [28%]
- 1/3 medial (grup III) [~3%]

## Etiologi :

- Trauma langsung → jatuh menghantam bahu, pukulan langsung ke bahu
- Trauma tidak langsung → jatuh bertumpu pada tangan
- Trauma jalan lahir
