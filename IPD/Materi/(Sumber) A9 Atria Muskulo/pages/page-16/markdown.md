# Duchenne Muscular Dystrophy

## Tata Laksana

### Farmakologis
- DMD: Kortikosteroid (mis. Prednisone, Deflazacort)
- BMD: Kortikosteroid (efikasi rendah)

### Suportif
- Fisioterapi → mencegah kontraktur
- Alat bantu jalan (mis. walker, kursi roda)
- Bedah (koreksi kontraktur dan skoliosis)
