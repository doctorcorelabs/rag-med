# Dislokasi Bahu

# Manifestasi Klinis

|  Tipe Dislokasi Bahu | Manifestasi Klinis  |
| --- | --- |
|  Anterior | • Akromion menonjol
• Kontur bahu hilang
• Lengan dalam posisi **adduksi** dan **eksternal rotasi**  |
|  Posterior | • Bahu posterior menonjol, anterior rata
• Prosesus korakoid menonjol
• Lengan dalam posisi **adduksi** dan **internal rotasi**  |
|  Inferior | • Posisi lengan di atas kepala, tidak bisa adduksi
• Disfungsi neurologis → cedera N. axillaris  |
