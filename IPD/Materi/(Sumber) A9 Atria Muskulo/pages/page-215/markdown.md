# Arthritis Septik

Klasifikasi infeksi sendi protesa:

|  Onset | Deskripsi  |
| --- | --- |
|  Early | <3 bulan post operasi implantasi
Pathogen utama : Staphylococcus aureus, Gram negatif  |
|  Delayed | 3-24 bulan post operasi implantasi
Pathogen utama : Staphylococcus aureus coagulase negative, Propionibacterium acnes  |
|  Late | >24 bulan post operasi implantasi
Pathogen utama : Penyebaran hematogen dari fokus infeksi lain  |
