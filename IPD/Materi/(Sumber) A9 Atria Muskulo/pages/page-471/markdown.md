# Tenosinovitis Supuratif

= Pyogenic flexor tenosynovitis

Definisi: Inflamasi akibat infeksi bakteri pada lapisan pembungkus tendon fleksor jari tangan (flexor tendon sheath)

## Etiologi:

- Infeksi bakteri → Staphylococcus aureus (40-75%), MRSA
- Mekanisme
- Trauma tembus ke pembungkus tendon
- Penyebaran dar: felon, arthritis septik, infeksi tempat lain
