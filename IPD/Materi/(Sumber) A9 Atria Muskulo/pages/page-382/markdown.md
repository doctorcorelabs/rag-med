# Cedera Meniskus

## Klasifikasi:
- Dibagi menjadi cedera meniskus medial dan lateral

|  Parameter | Meniskus medial | Meniskus lateral  |
| --- | --- | --- |
|  Bentuk | C-shaped | O-shaped  |
|  Ketebalan | Lebih tipis | Lebih tebal  |
|  Mobilitas | Kurang mobile | Lebih mobile  |
