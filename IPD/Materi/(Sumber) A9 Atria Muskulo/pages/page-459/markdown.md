# Osteomielitis

## Patofisiologi

- Inflamasi → Kongesti vaskularisasi tulang akibat infeksi → ↑ tekanan intraoseus → nyeri tulang &amp; trombosis vaskular
- Supurasi → Infeksi menyebar ke subperiosteum → fokus infeksi (abses Brodie)
- Sekuestrum → kematian tulang akibat infeksi dan aliran darah yang rendah
- Involukrum → pembentukan tulang baru
- Resolusi/Komplikasi
- Resolusi → progresi infeksi berhenti
- Komplikasi → infeksi menyebar ke jaringan lunak → sinus tract → supurasi
