# Take Home Points

## Spondilolisthesis

- Nomenklatur:
- Spondilolisis → stress fracture pars interarticularis vertebra
- Spondilolistesis → pergeseran vertebra; biasanya pasien spondilolisthesis juga mengalami spondilolisis
- Spondilosis → osterarthritis degeneratif vertebra
- Manifestasi klinis → Nyeri menjalar ke kaki
- Pemeriksaan penunjang → X-ray → pergeseran vertebra &amp; decapitated scottie dog sign (bila ada spondilolisis juga)
- Tata laksana pembedahan → Fusi vertebra
