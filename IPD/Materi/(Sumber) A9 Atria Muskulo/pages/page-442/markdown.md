# Ankle Sprain

## Tata Laksana:
- Rest, ice, compression, elevation (RICE)
- NSAID
- Latihan → penting untuk pemulihan
- Latihan ROM → plantarfleksi, dorsofleksi, dan gerakan melingkar kaki
- Dilakukan setelah nyeri akut membaik
- Imobilisasi: Terutama untuk sprain grade II &amp; III
- Bedah: Hanya dilakukan untuk sprain berat dengan risiko cedera berulang (atlet)
