## Skoliosis

- Etiologi → Kongenital, postural, patologis
- Manifestasi klinis
- Kurvatura lateral dari vertebra
- Pemeriksaan khusus → Adam's forward bend test
- Pemeriksaan penunjang → X ray vertebra + ukur Cobb angle
- Tata laksana → Observasi / brace / bedah
