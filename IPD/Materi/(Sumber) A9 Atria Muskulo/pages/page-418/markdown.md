# Sindrom Kompartemen

## Patofisiologi
- Fascia otot bersifat kaku (non-compliant)
- ↓ Ruangan dan ↑ volume cairan intrakompartemen otot → tekanan tinggi → gangguan perfusi → iskemi → nekrosis

## Klasifikasi
- Akut → kasus trauma, merupakan emergensi bedah
- Kronis → akibat aktivitas berlebih, bukan emergensi

Tekanan intrakompartemen (ICP) normal adalah 10mmHg
Pada sindroma kompartemen akut ICP dapat mencapai ≥30 mmHg
