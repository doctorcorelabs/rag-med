# Artritis Gout

## Manifestasi Klinis

- Fase interkritikal
- Asimtomatik → gejala akut menghilang
- Durasi: biasanya 2 tahun, kemudian akan muncul flare berulang

- Fase kronik: pembentukan tophus
- Pembengkakan keras → berisi kristal urat disertai kerusakan jaringan sekitar (deformitas)
- Biasanya tidak nyeri → namun bila meradang dapat menjadi nyeri
- Dapat menonjol dari kulit → berwarna keputihan
