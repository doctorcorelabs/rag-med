# Sarkoma Ewing

## Manifestasi Klinis:

- Gejala lokal:
- **Nyeri tulang**: intermiten, bertambah parah saat malam
- Kekakuan, bengkak tulang
- Fraktur patologis

- Gejala sistemik:
- Demam
- BB turun
- Gejala terkait metastasis:
- keterlibatan paru/pleura → suara napas asimetris, ronki
- Keterlibatan medula spinalis → ptekie/purpura (trombositopenia)
