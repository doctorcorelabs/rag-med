# Spondilosis

## Manifestasi Klinis:

- Pemeriksaan khusus:
- Spurling test → untuk radikulopati servikal
- Positif apabila terdapat nyeri dan parestesia pada penekanan leher
- Lasegue test (Straight leg raise test) → untuk radikulopati lumbal
- Positif apabila terdapat nyeri dan parestesia pada saat tungkai dinaikkan tanpa menekuk lutut
