# Spondilolisthesis

Definisi: Kondisi dimana vertebra tidak dapat mempertahankan strukturnya sehingga bergeser terhadap yang lain

## SPONDILO-

- Lisis → **stress fracture** pars interarticularis vertebra
- Listhesis → **pergeseran vertebra**; biasanya pasien spondilolisthesis juga mengalami spondilolisis
- Sis → **osteroarthritis** degeneratif vertebra
