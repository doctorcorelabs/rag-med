# Karsinoma Pankreas

|  Staging | Penyebaran  |
| --- | --- |
|  Stage I | Terbatas di **pankreas**  |
|  Stage II | Infiltrasi **duktus biliaris** dan sekitarnya; **limfonodus** -  |
|  Stage III | **Limfonodus +**  |
|  Stage IVA | **Metastasis organ terdekat**
(lambung, liver, diafragma, adrenal)  |
|  Stage IVB | **Metastasis jauh**  |

Pembedahan tidak dapat dilakukan apabila sudah mencapai stadium III dan/atau keterlibatan arteri mesenterika superior
