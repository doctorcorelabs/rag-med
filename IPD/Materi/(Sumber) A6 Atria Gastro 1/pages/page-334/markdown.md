# Kolesistitis Akut

|  Kriteria Diagnosis Kolesistitis Akut  |   |
| --- | --- |
|  Tanda lokal | • Murphy's sign
• Nyeri atau massa RUQ  |
|  Tanda sistemik | • Demam
• CRP meningkat
• Leukositosis  |
|  Temuan radiologis | • Temuan radiologis apapun yang menunjang kolesistitis akut  |

## Interpretasi

Diagnosis suspek: ≥1 lokal + ≥1 sistemik

Diagnosis definitif: ≥1 lokal + ≥1 sistemik + temuan radiologis
