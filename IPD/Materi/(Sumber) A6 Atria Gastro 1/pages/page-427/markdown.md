# Fissura Ani

- Definisi → Sobekan superfisial, distal dari linea dentata
- Etiologi → Tekanan tinggi pada anus
- Manifestasi klinis → Nyeri terutama saat BAB, berdarah
- Pemeriksaan penunjang → Tidak dibutuhkan pada fissura ani primer
- Tata laksana → Diet tinggi serat, pelunak tinja, analgetik
