# Varises Esofagus

## Tata Laksana

- Terapi Profilaksis → untuk **mencegah perdarahan varises**
- Primer → mencegah perdarahan pertama
- **Propranolol** 2 x 20 mg/hari
- Carvedilol 2 x 6,25-12,5 mg/hari
- Sekunder → mencegah perdarahan berikutnya
- Bila sudah terpasang TIPS tidak perlu pencegahan sekunder
- Ligasi + **Propranolol** → pilihan pertama
- TIPS → pilihan kedua
- Transplantasi hepar
