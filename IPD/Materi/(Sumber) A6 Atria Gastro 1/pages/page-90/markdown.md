# Akalasia

Definisi: Kegagalan otot polos untuk berelaksasi, sehingga menyebabkan **lower esophageal sphincter (LES) tertutup**

## Etiologi:

- Degenerasi pleksus Auerbach dan serat nervus vagus pada LES
