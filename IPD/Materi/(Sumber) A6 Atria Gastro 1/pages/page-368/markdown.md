# Pankreatitis Akut

## Pemeriksaan Penunjang:

- ↑ Enzim pankreas serum
- Lipase → ≥3x batas atas nilai normal → indikasi kuat pankreatitis akut
- Amilase
- ↑ ALP (alkalin fosfatase) dan bilirubin → pada etiologi batu empedu
- USG abdomen
- Batas pankreas tidak jelas
- Kumpulan cairan peripankreas
