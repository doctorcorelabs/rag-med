# Case Description

Pasien mengaku sering merasa panas di daerah dada dan terasa pahit di mulutnya sejak beberapa bulan lalu. Pasien merupakan seorang **perokok** dan **sering minum kopi hitam**. A. Striktur esofagus
B. Akalasia esofagus
