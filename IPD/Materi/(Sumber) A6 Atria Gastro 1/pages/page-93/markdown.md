# Akalasia

## Pemeriksaan Penunjang:

- Esofagogram barium (**barium swallow**) → pemeriksaan awal
- **Bird beak sign** atau **rat tail sign**
- Endoskopi (EGD)
- Eksklusi lesi premalignan dan malignan
- Manometri esofagus → **gold standard**
- Relaksasi tidak komplit dari LES saat menelan
- Tekanan LES tinggi
- Tidak ada gerakan peristaltik
