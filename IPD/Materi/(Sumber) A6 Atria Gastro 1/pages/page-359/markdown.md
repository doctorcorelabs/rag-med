# Kolangitis

|  Kriteria Diagnosis Kolangitis Akut  |   |
| --- | --- |
|  Tanda inflamasi | • Leukositosis, peningkatan LED dan CRP  |
|  Tanda kolestasis | • Jaundice
• Gangguan fungsi liver (peningkatan bilirubin, SGPT/SGOT, atau ALP)  |
|  Temuan radiologis (USG/CT/MRI) | • Dilatasi duktus koledokus (CBD)
• Temuan etiologi yang mendasari (koledokolitiasis, striktur, dll.)  |

# Interpretasi

Diagnosis suspek: ≥1 tanda inflamasi + ≥1 tanda kolestasis ATAU ≥1 temuan radiologis
Diagnosis definitif: ≥1 tanda inflamasi + ≥1 tanda kolestasis + ≥1 temuan radiologis

Number: Tokyo Guidelines, 2018
