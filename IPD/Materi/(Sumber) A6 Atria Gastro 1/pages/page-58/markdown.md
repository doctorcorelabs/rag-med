# Atresia Esofagus &amp; Fistula Trakea

## Etiologi:

- Kegagalan perkembangan dari foregut atau kesalahan diferensiasi mesoderm
- Terdapat 2 sindrom yang berkaitan dengan atresia esofagus, yaitu:

|  Sindrom VACTERL | Sindrom CHARGE  |
| --- | --- |
|  Vertebral anomaly | Coloboma  |
|  Anal atresia | Heart defects  |
|  Cardiac malformation | Atresia choanae  |
|  Tracheoesophageal fistula | Mental Retardation  |
|  Esophageal atresia | Genital hypoplasia  |
|  Renal anomaly | Ear abnormalities  |
|  Limb anomaly |   |
