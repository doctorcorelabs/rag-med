# Varises Esofagus

## Manifestasi Klinis

- Biasanya asimtomatis bila tidak terjadi perdarahan
- Hematemesis dan/atau melena
- Dapat menyebabkan syok hipovolemia bila perdarahan masif
- Tanda hipertensi porta
- Splenomegali
- Caput medusae → akibat distensi ligamentum teres hepatis
- Hemoroid
