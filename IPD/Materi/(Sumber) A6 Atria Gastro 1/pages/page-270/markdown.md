# Amebiasis

## Patofisiologi

- Kista dewasa tertelan → di usus halus beubah menjadi trofozoit → migrasi ke kolon
- Trofozoit di kolon:
1. Tetap di lumen usus (asimtomatis)
2. Invasi mukosa usus (amebiasis intestinalis/disentri amoeba)
3. Invasi pembuluh darah → hepar (abses hepar), otak, paru
- Trofozoit membelah, menghasilkan kista → keluar bersama feses
