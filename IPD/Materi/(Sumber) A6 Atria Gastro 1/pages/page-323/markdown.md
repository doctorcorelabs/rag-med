# Kolelitiasis

## Tata Laksana

- Observasi → pada pasien asimtomatis
- Bila pasien kolik bilier (+)
- Nil per oral (NPO) → pasien dipuasakan
- NSAID → untuk mengurangi rasa nyeri
- Cairan IV → bila pasien dehidrasi/muntah persisten
- *Ursodeoxycholic acid (UDCA) 2x250 mg PO* → untuk batu radiolusen berukuran &lt;0,5 cm
- Pembedahan
- **Kolesistektomi elektif** → terapi definitif
- ESWL → bila tidak dapat dilakukan kolesistektomi
