# Atresia Esofagus &amp; Fistula Trakea

Definisi: Defek kongenital dimana esofagus bagian atas tidak terhubung sempurna dengan esofagus bagian bawah

Klasifikasi :

|  Klasifikasi | Deskripsi  |
| --- | --- |
|  Gross A | Atresia esofagus tanpa fistula trakeoesofageal  |
|  Gross B | Atresia esofagus + fistula trakeoesofageal proksimal dari atresia  |
|  Gross C | Atresia esofagus + fistula trakeoesofageal distal dari atresia  |
|  Gross D | Atresia esofagus + fistula trakeoesofageal proksimal dan distal  |
|  Gross E | H-type fistula trakeoesofageal tanpa atresia  |
