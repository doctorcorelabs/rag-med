# Stenosis Pilorus Hipertrofi

## Pemeriksaan Penunjang:

- Foto polos
- Tampak gas lambung besar (single bubble appearance)
- USG
- Dinding: tebal &gt;3 mm, panjang &gt;15 mm → indikasi stenosis piloris
- Temuan target sign, perlambatan pengosongan lambung
- Barium swallow
- Pemanjangan pilorus, perlambatan pengosongan lambung
- String sign, double-track sign, beak sign
- Endoskopi
