# RASIONALE

Keluhan lemas dan mudah lelah tiap menstruasi (perdarahan, etiologi anemia def besi) + konjungtiva anemis (+) disertai Hb 8,7 g/dL, MCV 72 fl, MCH 20 pg (Mikrositik Hipokrom) → Dx.

# ANEMIA MIKROSITIK HIPOKROM

A. Anemia mikrositik hipokromik e.c. suspek penyakit kronik (adanya underlying diseases)
B. Anemia mikrositik hipokromik e.c. suspek defisiensi B12 (MCV &gt;100 fl)
C. Anemia makrositik hiperkromik e.c. suspek defisiensi B9 (MCV &gt;100 fl)
D. Anemia mikrositik hipokromik e.c. suspek defisiensi besi
E. Anemia mikrositik hiperkromik e.c. suspek defisiensi B9 (MCV &gt;100 fl)
