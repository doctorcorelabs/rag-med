### Analitik

|  Pembeda | Cross Sectional | Case Control | Cohort  |   |
| --- | --- | --- | --- | --- |
|   |   |   |  Retrospektif | Prospektif  |
|  Identifikasi | Saat itu juga, observasi dilakukan 1x saat pengumpulan data | Dimulai dari identifikasi antara kasus dan control, kemudian ditlusuri factor resikonya | Pencarian data melalui rekam medis | Pencarian data di masa yang akan datang (follow up, periode tertentu)  |
|  Tujuan | Prevalensi, proporsi | Causal association, probabilitas | Insiden, prognosis, natural history of the disease  |   |
|  Kelebihan | Mudah dan murah, sederhana | Biasanya untuk penyakit yang jumlah kasusnya sedikit | Menghasilkan data temporal relationship yang adekuat dan observasi yang seragam  |   |
|  Kekurangan | Temporal relationship yang tendah, bias diagnosis | Bias recall | Permasalahan etik, waktu lama, mahal, bias follow up  |   |
|  Hubungan | Prevalence ratio (PR) | Odd ratio (OR) / 1 | Relative Risk (RR)  |   |
