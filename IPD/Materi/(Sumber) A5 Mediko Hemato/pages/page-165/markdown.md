# RATIONALE

Wanita 60 tahun keluhan benjolan di leher yang semakin membesar disertai berat badan turun dan keringat malam hari (B-symptoms) + Pemfis massa soliter di regio colli, konsistensi kenyal, mobile, tidak ada nyeri tekan + biopsi didapatkan gambaran Reed-Stenberg cell → Dx.

# LIMFOMA HODGKIN

A. Limfangitis (tidak tepat)
B. Limfadenopati akut (tidak tepat)
C. Limfadenopati tuberkulosis (adanya riwayat TB atau keluhan khas TB)
D. Limfoma Hodgkin
E. Limfoma Non-Hodgkin (hasil biopsi gambaran starry sky)

|  Hodgkin | Non-hodgkin  |
| --- | --- |
|  B-symptoms: low-grade fever, BB turun, keringat malam  |   |
|  Keterlibatan nodus limfa soliter dan terlokalisir, menyebar contiguous (menyebar ke nodus atau jaringan ekstranodus sekitar) | Keterlibatan nodus limfatik yang multipel, menyebar non-contiguous  |
|  Sel reed-sternberg positif | Sebagian besar sel B, beberapa turunan dari sel T  |
|  Distribusi bimodal (pada dewasa muda dan >55 tahun) | Dapat terjadi pada anak-anak dan dewasa  |
|  Berhubungan dengan EBV | Berhubungan dengan penyakit autoimun dan infeksi virus (EBV, HIV, HTLV)  |
