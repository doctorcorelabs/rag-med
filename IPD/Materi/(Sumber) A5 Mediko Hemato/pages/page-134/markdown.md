# DISSEMINATED INTRAVASCULAR COAGULATION

## FAKTOR RISIKO

- Komplikasi obstetric (post partum)
- Trauma/jejas jaringan
- Reaksi alergi: bisa ular, toxic shock syndrome
- Keganasan
- Sepsis
- Gangguan vaskular: malformasi vaskular, anurisma

## DIAGNOSIS

- Bleeding time ( &gt;&gt; ), clotting time ( &gt;&gt; ), PT ( &gt;&gt; ), APTT ( &gt;&gt; ), platelet count ( &lt;&lt; ), fibrinogen ( &lt;&lt; )
- Tidak ada gold standard → dengan skor
- Fibrin degradation product (FDP) ( ☐ )
- D-dimer ( &gt;&gt; )

## TATALAKSANA

Terapi sesuai penyebab dan suportif. Transfusi komponen darah sesuai indikasi.
