# TATALAKSANA

- Terapi suportif → hidrasi, diet, antipiretik
- Fisioterapi dada
- Antibiotik empiris
- ✓ Klindamisin 3 x 600 mg IV (hari 1) → 4 x 150-300 mg PO (hari berikutnya); tatalaksana hingga terjadi resolusi pada foto thorax (3-6 minggu)
