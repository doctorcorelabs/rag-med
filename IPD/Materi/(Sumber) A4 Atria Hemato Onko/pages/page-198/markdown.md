# Hemolytic Uremic Syndrome (HUS)

## Manifestasi Klinis:

- Shigellosis: diare berdarah 5-10 hari sebelum muncul gejala HUS
- Trias HUS:
- Gejala trombositopenia: petekie, purpura, perdarahan mukosa
- Anemia hemolitik: konjungtiva pucat, ikterus
- Gangguan ginjal: hematuria, proteinuria, oliguria/anuria (AKI)
- Dapat terjadi gejala neurologis (mis. kejang, stroke, penurunan kesadaran), meskipun jarang

Berbeda dengan TTP, HUS umumnya tidak disertai demam
