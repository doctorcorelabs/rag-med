# Malaria

## Manifestasi Klinis

- Malaria berat → biasanya pada malaria tropica

|  Sistem | Kriteria (Hanya perlu satu)  |
| --- | --- |
|  Metabolik | Asidosis metabolik  |
|   |  Hiperlaktemia (asam laktat >5 mmol/L)  |
|   |  Hipoglikemia (gula darah <40 mg/dL)  |
|  Kardiovaskuler | Syok (CRT >3 detik, sistol <80 mmHg pada dewasa, sistol <70 mmHg pada anak)  |
|  Respirasi | Edema paru  |
|  Urologi | Kreatinin >3 mg/dL, ureum >20 mg/L  |
|   |  Hemoglobinuria → Black water fever  |
