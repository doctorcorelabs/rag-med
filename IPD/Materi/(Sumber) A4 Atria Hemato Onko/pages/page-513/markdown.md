# Henoch-Schonlein Purpura

## Tata Laksana

- Umumnya bersifat **swasirna**
- Analgesik → mengurangi nyeri
- NSAID (mis. Ibuprofen)
- Parasetamol
- Steroid → terutama bila nyeri tidak berkurang dengan analgesik
- **Prednison** tapering off, dosis awal 1-2 mg/kgBB/hari (max. 60-80 mg/hari) selama 1-2 minggu
