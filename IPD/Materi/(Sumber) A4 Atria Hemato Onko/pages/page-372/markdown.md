# Infeksi Dengue

## Manifestasi Klinis

- Perjalanan infeksi Dengue secara klinis terbagi menjadi 3 fase:

|  Parameter | Fase Febris | Fase Kritis | Fase Konvalesens  |
| --- | --- | --- | --- |
|  Durasi | 3 – 7 hari | 1 – 2 hari | 2 – 4 hari  |
|  Gejala | Demam tinggi | Demam mulai turun | Tanda kebocoran plasma menghilang  |
|   |  Gejala penyerta
(nyeri kepala, nyeri otot, nyeri retroorbita) | Tanda kebocoran plasma
(efusi pleura atau ascites) | Tanda vital stabil  |
|   |   |  Ht meningkat
(umumnya ≥20% baseline) | Dapat muncul ruam kulit yang gatal (convalescent rash)  |
