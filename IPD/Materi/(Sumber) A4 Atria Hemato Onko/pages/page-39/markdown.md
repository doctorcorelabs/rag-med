# Thalassemia β

Bentuk-bentuk thalassemia β antara lain

|  Jenis | Ganggguan  |
| --- | --- |
|  β minor | 1 alel defektif (β/β* ATAU β/β⁰)  |
|  β intermedia | 2 alel defektif (β⁺/β⁺)  |
|  β mayor (anemia Cooley) | 2 alel defektif (β⁰/β⁰ ATAU β⁰/β⁺)  |

Thalassemia β intermedia sulit dibedakan dengan thalassemia β minor maupun β mayor
