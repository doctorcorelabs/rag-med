# Polisitemia

|  Jenis Polisitemia |   | Penyebab  |
| --- | --- | --- |
|  Polisitemia Relatif |   | Muntah, diare, penggunaan diuretik  |
|  Polisitemia Absolut | Primer | Mutasi yang menyebabkan proliferasi abnormal dari sel progenitor, misalnya **polisitemia vera** atau neoplasma mieloproliferatif lainnya  |
|   |  Sekunder | Peningkatan eritropoetin, bisa akibat **hipoksia** (tinggal di pegunungan) atau tumor yang memproduksi eritropoietin  |
|   |  Neonatal | Transfusi dari ibu ke anak, bisa akibat pemotongan tali pusat yang telat atau bayi yang diletakkan pada posisi lebih rendah dibandingkan ibunya setelah dilahirkan  |
