# Chagas Disease

## Manifestasi Klinis:

- Masa inkubasi (1-2 minggu)
- Fase akut (8-12 minggu) → mayoritas asimtomatis
- Gejala non-spesifik: demam, malaise, anoreksia
- Chagoma: pembengkakan di area gigitan → wajah dan ekstremitas
- Romana sign: pembengkakan area kelopak mata yang tidak nyeri
- Fase indeterminate (bertahun-tahun) → pasien asimptomatis
- Fase kronik
- Gangguan KV: RBBB, kardiomiopati dilatasi
- Gangguan GIT: megaaesofagus, akalasia
