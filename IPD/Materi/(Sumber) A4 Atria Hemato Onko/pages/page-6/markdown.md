# Anemia Defisiensi Besi

Definisi: Anemia akibat kurangnya zat besi untuk eritropoesis, sehingga pembentukan hemoglobin berkurang

## Etiologi:

- Kurang asupan besi (rendah vit. C, rendah daging, banyak teh/coklat)
- Kehilangan besi akibat perdarahan kronik (ulkus peptik, kanker, hemoroid, **menometroragia**) → penyebab tersering
- Kebutuhan besi meningkat (masa pertumbuhan, hamil)
- Gangguan absorbsi besi (post-gastrektomi, celiac disease)
