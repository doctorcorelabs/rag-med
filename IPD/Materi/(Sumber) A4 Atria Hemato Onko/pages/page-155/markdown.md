# Hemofilia

## Tata Laksana

- Terapi pengganti faktor pembekuan
- Konsentrat faktor → tata laksana definitif
- Hemofilia A → konsentrat faktor VIII
- Hemofilia B → konsentrat faktor IX
- Kriopresipitat → alternatif
- Mengandung faktor VIII, fibrinogen, vWF
- Tidak dapat diberikan pada Hemofilia B
- Fresh frozen plasma (FFP) → alternatif
- Mengandung seluruh faktor, namun dalam konsentrasi rendah
