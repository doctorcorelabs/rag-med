# Sepsis

## Skoring:

- Menggunakan skor qSOFA (quick sequential organ failure assessment):
- Tujuan: **mengidentifikasi risiko seseorang mengalami sepsis pada pasien di luar ICU**
- Komponen skoring:

|  Gangguan | Skor  |
| --- | --- |
|  GCS <15 | 1  |
|  RR ≥22 kali/menit | 1  |
|  TD sistolik ≤100 mmHg | 1  |

## Hasil:

- 0-1 poin: risiko rendah sepsis
- 2-3 poin: risiko tinggi sepsis
