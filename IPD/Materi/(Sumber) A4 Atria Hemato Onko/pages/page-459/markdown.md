# Pemeriksaan Penunjang

- Virologi
- Indikasi
- Usia &lt;18 bulan, karena kemungkinan antibodi maternal masih ada → serologi tidak dilakukan
- Infeksi HIV primer (klinis seperti infeksi virus pada umumnya)
- Hasil pemeriksaan serologi indeterminate atau inkonklusif
- Metode: PCR
- Monitoring dapat dilakukan dengan menghitung viral load dan hitung CD4
