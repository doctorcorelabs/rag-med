# Henoch-Schonlein Purpura

Kriteria Diagnosis berdasarkan EULAR/PRINTO/PRES

|  Kriteria | Deskripsi  |
| --- | --- |
|  Wajib | Palpable purpura atau petekie, umumnya pada ekstremitas bawah  |
|  Min.
1 dari 4 | Nyeri perut berat dengan onset akut  |
|   |  Arthritis atau arthralgia dengan onset akut  |
|   |  Histopatologi menunjukkan gambaran **vaskulitis** atau glomerulonefritis dengan dominan deposit IgA  |
|   |  Ggn. ginjal → hematuria atau **proteinuria**  |
