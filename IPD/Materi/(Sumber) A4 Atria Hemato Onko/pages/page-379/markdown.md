# Kriteria Diagnosis

## DENGUE FEVER (DF)

### Kriteria Diagnosis
1. Demam akut
2. Min. 2 gejala:
- Ruam
- Manifestasi perdarahan (+)
- Tidak ada fase kritis
- Peningkatan hematokrit 5-10%
3. IgM/IgG Dengue atau NS1

## DENGUE HEMORRHAGIC FEVER (DHF)

### Kriteria Diagnosis
1. Demam akut
2. Manifestasi perdarahan
3. Bukti kebocoran plasma
- Peningkatan Ht &gt;20%
- Efusi pleura
- Asites
- Hipoalbuminemia
4. Trombositopenia (&lt;100.000/mikroL)

## DENGUE SHOCK SYNDROME (DSS)

### Kriteria Diagnosis
1. Kriteria DHF
2. Tanda gangguan hemodinamik
- Takikardia
- Ekstremitas dingin
- Nadi lemah / hilang
- Hipotensi
- Jarak sistol - diastol (pulse pressure) &lt;20 mmHg
