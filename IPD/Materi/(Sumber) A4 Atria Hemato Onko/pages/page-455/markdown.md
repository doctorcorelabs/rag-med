# HIV

## Strategi Tes HIV (WHO)

- Menggunakan EIA atau rapid test
- Terdapat 3 strategi tes HIV dengan tujuan berbeda:

|  STRATEGI I | Tujuan: pemeriksaan darah dan organ untuk donor atau transplantasi  |
| --- | --- |
|   |  Memerlukan hanya 1 jenis reagen (reagen A1)  |
|  STRATEGI II | Tujuan: surveilans sentinel (studi epidemiologi), skrining acak, misalnya pada pasien yang datang ANC atau menderita IMS  |
|   |  Memerlukan 2 jenis reagen (reagen A1 dan A2)  |
|  STRATEGI III | Tujuan: diagnosis individu, misalnya pada suspek bergajala  |
|   |  Memerlukan 3 jenis reagen (reagen A1, A2, dan A3)  |
