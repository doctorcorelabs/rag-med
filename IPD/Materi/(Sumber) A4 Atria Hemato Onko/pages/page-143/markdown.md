# von Willebrand Disease

## Tata Laksana:

- vWD genetik
- Desmopressin (DDAVP) → menstimulasi pelepasan vWF dari endotel
- Diberikan pada tipe I dan II
- Tidak efektif untuk tipe III → vWF sama sekali tidak terbentuk
- Konsentrat vWF dan faktor VIII → pada kasus perdarahan berat
- Kriopresipitat → bila tidak tersedia konsentrat vWF dan faktor VIII
- Mengandung faktor VIII, fibrinogen, dan vWF
