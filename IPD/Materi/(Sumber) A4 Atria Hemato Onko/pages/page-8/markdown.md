# Anemia Defisiensi Besi

## Manifestasi Klinis

- Gejala anemia
- Malaise/letargi → lemas
- Konjungtiva dan wajah pucat
- Kelainan jantung: takikardia, kardiomiopati → pada anemia berat
- Koilonikia → kuku sendok
- Pica → mengonsumsi sesuatu tanpa nilai nutrisi
- Cheilitis angularis → fissura pada ujung bibir
- Glossitis atrofi → papil lidah atrofi (menghilang) sehingga lidah tampak licin
