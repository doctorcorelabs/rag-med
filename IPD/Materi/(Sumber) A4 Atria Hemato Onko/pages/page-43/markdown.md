# Thalassemia

## Pemeriksaan Penunjang

- Darah rutin
- Anemia mikrositik hipokrom
- RDW rendah (pada ADB RDW tinggi)
- Indeks Mentzer &lt;13 (pada ADB &gt;13)
- Rumus: $\frac{MCV}{\text{jumlah RBC (dalam juta/mikroL)}}$
- Tanda hemolisis
- Retikulosit ↑
- Bilirubin indirek ↑

Anemia mikrositik hipokrom disertai hemolisis → PIKIRKAN Thalassemia!
