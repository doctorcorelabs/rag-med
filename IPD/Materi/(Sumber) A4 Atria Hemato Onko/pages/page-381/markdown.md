# Tata Laksana Dengue

Dengue **DENGAN** warning signs (DHF grade I-II)

- Periksa Ht baseline
- Beri cairan kristaloid (RL/NS) dengan skema:

```
Mulai dengan
5-7 mL/kg/jam
selama 1-2 jam
→ Lanjutkan
3-5 mL/kg/jam
selama 2-4 jam
→ Lanjutkan
2-3 mL/kg/jam
kemudian cek Ht
```

- Monitoring Ht
- Ht tetap → lanjutkan 2-3 mL/kg/jam selama 2-4 jam
- Ht meningkat → 5-10 mL/kg/jam selama 1-2 jam
- Ht menurun → kurangi perlahan

Number: WHO, 2012 &amp; PNPK Dengue Kemenkes RI 2021
