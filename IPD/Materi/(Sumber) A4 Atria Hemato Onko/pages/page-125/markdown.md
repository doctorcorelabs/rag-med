# Case Description

Pada PF tampak konjungtiva pucat dan teraba hepatosplenomegali. Wajah anak pucat dan tulang pipi anak tampak prominen. Pemeriksaan lab didapatkan Hb 5,8 g/dL, MCV 62 fL, MCH 23 pg. Hapusan darah tepi tampak adanya sel target dan dakriosit.
