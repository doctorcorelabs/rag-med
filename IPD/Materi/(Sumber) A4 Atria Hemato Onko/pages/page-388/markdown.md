# Malaria

Definisi: Infeksi parasit *plasmodium* didalam eritrosit, umumnya ditandai dengan gejala utama berupa *menggiggil* yang disertai *demam*

## Etiologi:

- Vektor → nyamuk *Anopheles betina*
- Patogen → *Plasmodium sp.*

|  Etiologi | Nama Penyakit  |
| --- | --- |
|  P. falciparum | Malaria tropica  |
|  P. vivax | Malaria tertiana  |
|  P. ovale  |   |
|  P. malariae | Malaria kuartana  |
