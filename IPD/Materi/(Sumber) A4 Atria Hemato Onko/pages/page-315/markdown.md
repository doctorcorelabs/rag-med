# Leukemia Mieloid Kronis

## Pemeriksaan Penunjang

### Darah lengkap
- Leukositosis berat (hingga &gt;500.000/mm³)
- Trombositosis (hingga &gt;600.000/mm³)
- Basofilia dan eosinophilia

### Apusan darah tepi
- Dominan granulosit, mielosit, dan metamielosit
- Sel blas → menentukan fase penyakit

|  Fase CML | % Sel blas pada darah tepi  |
| --- | --- |
|  Kronis | <10%  |
|  Akselerasi | 10-19%  |
|  Krisis blas | ≥20%  |
