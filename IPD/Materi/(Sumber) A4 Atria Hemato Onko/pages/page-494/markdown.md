# SLE

## Pemeriksaan Penunjang

- Antibodi anti-nukleus
- ANA: antibodi paling sensitif (95%), namun tidak spesifik untuk SLE
- Anti-dsDNA: antibodi yang cukup spesifik untuk SLE
- Anti-Sm (Smith): antibodi paling spesifik untuk SLE
- Penanda inflamasi ↑ (mis. LED, CRP)
- Sitopenia: anemia hemolitik, leukopenia, trombositopenia
- Gangguan fungsi ginjal → ureum dan kreatinin ↑↑
