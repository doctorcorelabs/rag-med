# Limfoma Non-Hodgkin

## Pemeriksaan Penunjang

- Pemeriksaan darah
- Leukositosis
- Hiperkalsemia (akibat produksi vitamin D berlebih, salah satu manifestasi sindrom paraneoplastik)
- Histopatologi → tes konfirmasi
- Sampel dari KGB
- Ditemukan proliferasi limfosit
- Aspirasi sumsum tulang: untuk menentukan staging penyakit
- Imunohistokimia: sel B (CD₂₀ (+)), sel T (CD₃ (+))
