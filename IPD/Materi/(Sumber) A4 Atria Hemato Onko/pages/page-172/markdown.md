# Immune Thrombocytopenic Purpura (ITP)

## Manifestasi Klinis

- Perdarahan berupa ekimosis, petekie dan purpura
- Derajat dan frekuensi perdarahan tergantung jumlah trombosit

|  Trombosit | Manifestasi  |
| --- | --- |
|  >50.000 | Asimtomatis  |
|  30.000-50.000 | Hematom/luka memar  |
|  10.000-30.000 | Perdarahan spontan mungkin terjadi, menoragia, perdarahan yang memanjang bila luka  |
|  <10.000 | Perdarahan mukosa spontan (mis. epistaksis, melena), risiko perdarahan SSP  |
