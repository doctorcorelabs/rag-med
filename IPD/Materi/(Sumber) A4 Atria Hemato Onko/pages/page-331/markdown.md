# Case Description

An. Bishop, laki-laki usia 5 tahun dibawa berobat ke dokter karena sering mimisan dan gusi berdarah saat sikat gigi sejak 2 minggu lalu. Anak juga cenderung lemas dan sering demam sejak beberapa bulan lalu. Pada pemeriksaan fisik dijumpai konjungtiva anemis dan hepatosplenomegali. Pemeriksaan lab didapatkan Hb 9,4 g/dL, Ht 30%, leukosit 58.000/mikroL, dan trombosit 144.000/mikroL. Biopsi sumsum tulang didapatkan limfoblas 30%.
