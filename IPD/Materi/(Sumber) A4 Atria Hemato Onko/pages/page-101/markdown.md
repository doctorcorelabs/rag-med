# Anemia Defisiensi B₉

## Pemeriksaan Penunjang

- Pemeriksaan darah
- Anemia makrositik (MCV &gt;100 fL)
- Pemeriksaan kadar asam folat serum
- &gt;4 ng/mL
- Normal, tidak membutuhkan pemeriksaan tambahan
- 2-4 ng/mL
- Borderline, memerlukan pemeriksaan lanjutan (MMA dan homosistein)
- &lt;2 ng/mL
- Rendah, dapat dilanjutkan tes MMA dan homosistein
