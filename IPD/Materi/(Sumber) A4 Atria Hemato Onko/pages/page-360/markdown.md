# Toksoplasmosis

## Manifestasi Klinis:

- Pada pasien **imunokompeten** → umumnya **asimtomatis**
- Limfadenopati servikal
- **Flu-like syndrome**: demam, keringat malam, mialgia, malaise
- Pada pasien **imunodefisiensi** → muncul **gejala CNS**
- Retinokoroiditis, pneumonitis, ensefalitis, meningitis, hemiparesis
- Pada pasien **AIDS (imunodefisiensi berat)** → perubahan status mental, lesi masa intraserebri, reaktivasi infeksi kronis
