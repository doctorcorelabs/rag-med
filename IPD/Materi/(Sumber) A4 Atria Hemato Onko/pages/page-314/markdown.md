# Leukemia Mieloid Kronis

## Manifestasi Klinis:

Perjalanan penyakit CML dibagi menjadi 3 fase:

- Fase krisis blas (blas ≥20%)
- Fase terakhir dari CML
- Gejala sama dengan leukemia akut (anemia, trombositopenia, dll.)
- Dapat terjadi dalam 2 bentuk:
- Krisis blas myeloid → AML (2/3 kasus)
- Krisis blas limfoid → ALL (1/3 kasus)
