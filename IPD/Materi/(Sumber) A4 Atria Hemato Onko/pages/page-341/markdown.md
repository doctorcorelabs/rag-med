# Chagas Disease

## Pemeriksaan Penunjang

### Fase akut

- Apusan darah tebal &amp; tipis → visualisasi tripomastigot dengan pewarnaan Giemsa
- Uji aglutinasi kartu
- PCR → dilakukan bila hasil apusan darah negatif namun penyakit Chagas masih dicurigai

### Fase kronik

- ELISA → deteksi IgG Trypanosoma cruzi
- Ekokardiografi → mencari gangguan jantung
- Barium swallow → deteksi skalasia
