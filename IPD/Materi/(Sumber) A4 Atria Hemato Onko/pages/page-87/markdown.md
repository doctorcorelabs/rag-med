# Inkompatibilitas Rhesus

## Patofisiologi:

- Kehamilan pertama:
- Ibu dengan Rh-negatif membentuk antibodi IgM maternal terhadap antigen D pada anak dengan Rh-positif
- IgM anti-D tidak dapat menembus sawar darah-plasenta

- Kehamilan kedua:
- Terjadi serokonversi IgM anti-D menjadi IgG anti-D yang dapat menembus sawar darah-plasenta
- Terjadi hemolisis berat → dapat terjadi hydrops fetalis
