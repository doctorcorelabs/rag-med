# Hemofilia

## Pemeriksaan Penunjang

- Aktivitas faktor koagulasi → penegakan diagnosis
- Hemofilia A → aktivitas faktor VIII &lt;40% atau &lt;0,4 IU/mL
- Hemofilia B → aktivitas faktor IX &lt;40% atau &lt;0,4 IU/mL
- Pemeriksaan genetik → penegakan diagnosis definitif
- Dapat mendeteksi wanita karier
