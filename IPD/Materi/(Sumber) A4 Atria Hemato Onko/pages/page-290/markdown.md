# Leukemia Myeloid Akut

## Manifestasi Klinis Leukemia Akut (AML &amp; ALL)

- Gejala anemia → Lemas, tidak bersemangat
- Gejala trombositopenia → Epistaksis, perdarahan gusi, petekie
- Gejala leukopeni → rentan infeksi
- Hepatosplenomegali
- Akibat infiltrasi sel blas, insidensi pada ALL &gt; AML
- Sindrom leukostasis
- Pengentalan darah akibat jumlah leukosit yang terlalu banyak (biasanya pada jumlah &gt;100.000/mm³)
