# Hemolytic Uremic Syndrome (HUS)

## Pemeriksaan Penunjang:

- Pemeriksaan darah
- Trombositopenia (&lt;40.000/mm³)
- Anemia (Hb &lt;9 g/dL)
- Leukositosis (~40.000-50.000/mm³)
- ↑ Retikulosit
- Peningkatan BUN dan kreatinin
- Profil koagulasi
- BT memanjang
- PT/aPTT cenderung normal (dapat sedikit meningkat)
- D-dimer cenderung normal (dapat sedikit meningkat)
