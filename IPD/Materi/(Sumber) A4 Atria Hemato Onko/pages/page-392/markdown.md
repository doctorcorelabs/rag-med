# Malaria

## Manifestasi Klinis

- Malaria berat → biasanya pada malaria tropica

|  Sistem | Kriteria (Hanya perlu satu)  |
| --- | --- |
|  Neurologi | Penurunan kesadaran GCS <11  |
|   |  Kejang >2 kali dalam 24 jam  |
|  Hematologi | Jaundice (bilirubin >3 mg/dL)  |
|   |  Perdarahan spontan  |
|   |  Anemia berat (dewasa <7 g/dL; anak <5 g/dL pada daerah endemis berat, <7 g/dL pada daerah endemis rendah-sedang)  |
|   |  Hiperparasitemia (>100.000 parasite/mikroL pada daerah endemis rendah, >200.000/mikroL pada endemis tinggi)  |
