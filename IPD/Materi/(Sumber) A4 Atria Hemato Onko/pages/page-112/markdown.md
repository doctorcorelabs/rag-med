# Anemia Defisiensi B₁₂

## Pemeriksaan Penunjang

- Pemeriksaan darah
- Anemia makrositik (MCV &gt;100 fL)
- Pemeriksaan antibodi pada anemia pernisiosa, antara lain:
- Antibodi terhadap faktor intrinsik (IF)
- Antibodi terhadap sel parietal
- Bila negatif, dilanjutkan pemeriksaan tes Schilling
