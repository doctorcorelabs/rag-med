# Leukemia Limfoblastik Akut

## Manifestasi Klinis ALL

- Demam tanpa infeksi → tidak terjadi pada AML
- Limfadenopati yang tidak nyeri, persisten dan progresif
- Nyeri tulang → cara berjalan aneh
- Meningeal leukemia → tanda &amp; gejala meningitis (mis. Kaku kuduk dan nyeri kepala) akibat infiltrasi sel blas
- Obstruksi saluran napas → akibat infiltrasi sel blas pada timus
- Pembesaran testis
