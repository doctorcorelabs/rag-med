# Leukemia Mieloid Kronis

## Manifestasi Klinis:

Perjalanan penyakit CML dibagi menjadi 3 fase:

- Fase kronis (blas &lt;10%) → asimptomatis, dapat bertahan beberapa tahun
- Gejala konstitusional: penurunan BB, demam, keringat malam
- Splenomegali
- Fase akselerasi (blas 10-19%)
- Tanda anemia: lemas, konjungtiva pucat
- Leukostasis: banyaknya leukosit menyebabkan sumbatan pembuluh darah → terjadi iskemi jaringan
- Splenomegali ekstrim → pertanda krisis blas
