# Hernia Femoralis

Klasifikasi:

|  Parameter | Hernia Inguinalis Indirek (Lateralis) | Hernia Inguinalis Direk (Medialis)  |
| --- | --- | --- |
|  Keluarnya usus | Melalui *prosesus vaginalis* | Melalui *kelemahan fascia transversalis*  |
|   |  Lateral dari pembuluh darah epigastrika inferior | Medial dari pembuluh darah epigastrika inferior (pada segitiga Hasselbach)  |
|  Etiologi | Kongenital: kegagalan menutupnya prosesus vaginalis | Mengangkat beban berat, biasa pada usia tua  |
|  Cincin Inguinalis | Melewati cincin inguinalis profunda dan superfisialis | Hanya melewati cincin inguinalis superfisialis  |
