# Take Home Points

## Ankilostomiasis

- Etiologi
- Ancylostoma duodenale &amp; Necator americanus → menyebabkan ankilostomiasis
- Ancylostoma caninum &amp; Ancylostoma braziliense → menyebabkan cutaneous larva migrans
- Stadium infektif → larva filariform
- Stadium diagnostik → telur / larva filariform (pada cutaneous larva migrans)
- Pemeriksaan feses → telur berdinding tipis dengan inti banyak
- Tata laksana lini pertama → Albendazole 400 mg dosis tunggal
