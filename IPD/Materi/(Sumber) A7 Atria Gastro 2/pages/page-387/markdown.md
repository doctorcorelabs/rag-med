# Strongyloidiasis

## Pemeriksaan Penunjang

- Pemeriksaan feses
- Mikroskopi feses: ditemukan larva rhabditiform → sensitivitas 21%
- Metode konsentrasi feses (Teknik Baermann) → sensitivitas 70-90%
- Pemeriksaan serologi (ELISA, IFAT) → sensitivitas tinggi, spesifisitas rendah
- PCR &amp; RT-PCR → spesifisitas 99%, sensitivitas sangat tinggi

Pada strongyloidiasis, jumlah cacing rendah dan output larva ireguler Sehingga sensitivitas pemeriksaan feses rendah
