# Penyakit Crohn

## Manifestasi Klinis:

- Ekstraintestinal
- Kulit → Pioderma gangrenosum, eritema nodosum
- Mata → uveitis, episkleritis
- Mulut → stomatitis aftosa
- Bilier → steatosis hepar, batu empedu, kolangitis
- Urinarius → nefrolitiasis, hidronefrosis, ISK
- Sendi → arthritis, ankylosing spondylitis
