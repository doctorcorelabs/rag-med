# Omfalokel vs Gastroskisis

|  Parameter | Omfalokel | Gastroskisis  |
| --- | --- | --- |
|  Lokasi | Tengah umbilikus | Periumbilikal dekstra  |
|  Kantong Pembungkus | Ada, berupa membran amnion dan peritoneum | Tidak ada kantong pembungkus  |
|  Tali Pusat | Ada, kaudal dari kantong hernia | Tidak ada  |
|  Anomali Lainnya | Sering terjadi bersamaan dengan anomali jantung | Jarang terjadi bersamaan dengan anomali lainnya  |
