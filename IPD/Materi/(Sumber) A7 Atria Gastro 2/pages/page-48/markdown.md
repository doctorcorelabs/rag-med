# Penyakit Hirschsprung

Diagnosis Banding

|  Penyakit | Ciri Khas  |
| --- | --- |
|  Hirschsprung | Aganglionosis pleksus Meissner dan Auerbach  |
|  Intestinal neuronal dysplasia | Hiperplasia pleksus submukosa (Meissner)  |
|  Ileus mekonium | Berhubungan dengan penyakit cystic fibrosis, enema menunjukkan gambaran mikrokolon  |
|  Hipotiroid kongenital | Ileus paralitik disertai dengan tanda hipotiroidisme kongenital lainnya (lemas, myxedema, makroglossia)  |
