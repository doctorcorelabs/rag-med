# Irritable Bowel Syndrome

## Pemeriksaan Penunjang → Tidak ada penunjang definitif

- Rujuk untuk Endoskopi bila terdapat gejala Alarm → untuk rule-out kanker kolon
- Usia onset &gt;50 tahun
- Hematochezia / melena
- Diare nocturnal
- Nyeri perut progresif
- Penurunan BB tanpa diinginkan
- Riwayat keluarga IBD atau kanker kolon
