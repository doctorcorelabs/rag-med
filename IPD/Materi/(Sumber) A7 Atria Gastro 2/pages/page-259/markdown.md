# Etiologi Ascites berdasarkan SAAG

|  SAAG ≥1,1 g/dL | SAAG <1,1 g/dL  |
| --- | --- |
|  Sirosis | Karsinoma  |
|  Hepatitis akibat alkohol | Tuberkulosis  |
|  Gagal jantung | Pankreatitis  |
|   | Sindrom nefrotik  |

SAAG secara tidak langsung menilai tekanan porta
Sehingga, SAAG tinggi = tekanan porta tinggi
