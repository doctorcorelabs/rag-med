# Irritable Bowel Syndrome

## Pemeriksaan Penunjang → Tidak ada penunjang definitif

- Diagnosis ditegakkan dengan **ROME IV**
- Bila diare:
- Fekalisis → untuk rule out infeksi
- Calprotectin / Laktoferin feses → untuk rule out IBD
- Calprotectin: &gt;50 mcg/g mengarah ke IBD
- Laktoferin: &gt;7,25 mg/g mengarah ke IBD
- Serologi untuk penyakit Celiac
