# Taeniasis

Definisi: Infestasi akibat cacing genus Taenia

Etiologi: Taenia saginata &amp; Taenia solium (cacing pita)

|  Parameter | Taenia Saginata | Taenia Solium  |
| --- | --- | --- |
|  Hospes sementara | Sapi | Babi  |
|  Stadium infektif | Sistiserkus bovis → menyebabkan taeniasis intestinalis | • Sistiserkus selulosa → menyebabkan taeniasis intestinalis
• Telur → menyebabkan sistiserkosis  |
|  Diagnosis | Telur/proglottid | Telur/proglottid  |
|  Sistiserkosis | Tidak | Ya  |
|  Jumlah uterus | 15-30 cabang dikotom | 5-10 cabang dendritik  |
