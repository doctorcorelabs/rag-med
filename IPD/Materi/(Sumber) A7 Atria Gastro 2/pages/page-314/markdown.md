# Kolera

## Tata Laksana:

- Resusitasi cairan → terapi terpenting pada kasus kolera
- Dehidrasi 5-10% → rehidrasi oral
- Dehidrasi &gt;10% → RL 100 ml/kg dalam 3 jam pertama
- Antibiotik
- Diberikan setelah resusitasi awal dan pasien dipastikan dapat menelan obat
- Pilihan antibiotik dapat dilihat di slide selanjutnya

Mortalitas kolera = 50-70 %
Dengan resusitasi cairan adekuat menjadi 0,5%
