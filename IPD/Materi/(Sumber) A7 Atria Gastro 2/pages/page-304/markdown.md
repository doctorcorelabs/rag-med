# Disentri Basiler vs Disentri Amoeba

|  Karakteristik | Shigellosis (Disentri Basiler) | Amebiasis (Disentri Amoeba)  |
| --- | --- | --- |
|  Frekuensi BAB | >10 kali/hari | 6-8 kali/hari  |
|  Volume BAB | Sedikit | Banyak  |
|  Bau | Tidak berbau | Bau busuk  |
|  Warna Feses | Merah terang | Merah gelap  |
|  pH feses | Basa | Asam  |
|  Kristal Charcot-Leyden | Tidak ada | Ada  |
|  Badan Piknotik | Tidak ada | Ada  |
|  Ghost cells | Ada | Tidak ada  |
|  Gold Standard | Kultur agar MacConkey | Mikroskopi feses  |
