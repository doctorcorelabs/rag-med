# Hernia Inguinalis

## Manifestasi Klinis

- Benjolan keluar masuk pada lipat paha
- Pada kasus hernia inguinalis lateralis dapat mencapai skrotum

## Zieman test (3-finger test)

- Hernia inguinalis lateralis → benjolan pada jari ke-2
- Hernia inguinalis medialis → benjolan pada jari ke-3
- Hernia femoralis → benjolan pada jari ke-4

## Finger test

- Hernia inguinalis lateralis → benjolan pada ujung jari
- Hernia inguinalis medialis → benjolan pada sisi medial jari
