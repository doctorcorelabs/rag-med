# Penyakit Crohn

## Pemeriksaan Penunjang:

- Darah lengkap → terutama dijumpai anemia
- Penanda inflamasi
- Terutama untuk membedakan IBD dan IBS
- Peningkatan **fecal calprotectin**, LED, dan CRP pada penyakit Crohn
- Serologi
- Untuk membedakan penyakit Crohn dan kolitis ulseratif
- Anti-Saccharomyces cerevisiae antibodies (ASCA) meningkat pada penyakit Crohn
- Perinuclear anti-neutrophil cytoplasmic antibodies (pANCA) meningkat pada colitis ulseratif
