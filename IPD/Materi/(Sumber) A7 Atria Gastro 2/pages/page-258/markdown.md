# Spontaneous Bacterial Peritonitis

## Pemeriksaan Penunjang

- Darah: leukositosis
- Parasentesis → dilakukan sebelum antibiotik diberikan
- Neutrofil ≥250 sel/mikroL
- Serum-ascites albumin gradient (SAAG) &gt;1,1
- Kultur untuk menentukan antibiotik terbaik

SAAG dihitung dengan rumus: Albumin serum – Albumin cairan ascites
