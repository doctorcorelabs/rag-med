# Demam Tifoid

## Pemeriksaan Penunjang:

- Leukopenia (dewasa) atau Leukositosis (anak-anak)
- Kultur
- Kultur darah → minggu 1
- Kultur feses → minggu 2-3 (sensitivitas &lt;50%)
- Kultur urin → minggu 2-4 (sensitivitas &lt;50%)
- Kultur sumsum tulang → dapat dilakukan pada minggu berapapun
- Merupakan **gold standard** dengan sensitivitas mencapai 90%
- Jarang digunakan karena prosedur invasif
