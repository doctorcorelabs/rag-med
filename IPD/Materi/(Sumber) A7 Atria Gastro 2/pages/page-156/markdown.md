# Kolitis Ulseratif

Derajat Keparahan

|  Derajat | Manifestasi  |
| --- | --- |
|  Ringan (mild) | Frekuensi BAB ≤4 kali/hari, dapat bercampur sedikit darah  |
|   |  Gejala ringan (tenesmus, nyeri perut, konstipasi), tidak ada gejala sistemik  |
|   |  Marker inflamasi (LED dan CRP) normal  |
|  Sedang (moderate) | Frekuensi BAB 4-6 kali/hari bercampur darah  |
|   |  Dapat dijumpai anemia ringan  |
|   |  Gejala sistemik biasanya minimal  |
|  Berat (severe) | Frekuensi BAB ≥6 kali/hari bercampur darah  |
|   |  Gejala sistemik berat (demam, takikardia, anemia berat)  |
|   |  Marker inflamasi (LED dan CRP) meningkat  |
