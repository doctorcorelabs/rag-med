# Apendisitis Akut

## Manifestasi Klinis:

- Nyeri abdomen bermigrasi → paling umum
- Dari periumbilikus (iritasi peritoneum visceral) ke RLQ (iritasi peritoneum parietal) dalam 12-24 jam
- Anoreksia (80%) → Jika nafsu makan normal, kemungkinan besar bukan appendisitis
- Mual muntah
