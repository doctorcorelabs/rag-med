# Giardiasis

- Etiologi → Giardia lamblia
- Manifestasi klinis → diare, steatorrhea
- Pemeriksaan penunjang → mikroskopi feses 3 sampel
- Tata laksana → Metronidazole 3x250-500 mg selama 5-10 hari
