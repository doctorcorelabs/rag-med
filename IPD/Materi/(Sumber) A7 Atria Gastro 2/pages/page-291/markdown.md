# Disentri Basiler

Definisi: Diare bakterial akibat bakteri Gram negatif spesies shigella

## Etiologi:

- Shigella → bakteri Gram negatif dan nonmotil, ada 4 serotipe
- Serotipe A → Shigella dysenteriae
- Serotipe B → Shigella flexneri
- Serotipe C → Shigella boydii
- Serotipe D → Shigella soneii
- Transmisi → fekal-oral
