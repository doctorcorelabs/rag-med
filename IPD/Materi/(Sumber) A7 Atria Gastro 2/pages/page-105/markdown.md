# Alergi Susu Sapi

## Pemeriksaan Penunjang:

- Uji cukit/tusuk kulit (skin prick test)
- Dilakukan di volar lengan bawah
- Uji kulit dapat negatif bila non-IgE mediated
- Uji IgE RAST (Radioallergosorbent test)
- Diperiksa apabila skin prick test tidak dapat dilakukan
- Positif apabila:
- Serum IgE spesifik &gt;5 kIU/L (anak ≤2 tahun)
- Serum IgE spesifik &gt;15 kIU/L (anak &gt;2 tahun)
