# Irritable Bowel Syndrome

## Definisi &amp; Kriteria Diagnosis ROME IV:

- **Nyeri perut rekuren** minimal 1x/minggu selama 3 bulan terakhir, yang berhbungan dengan dua atau lebih hal berikut:
- **Berhubungan dengan defekasi** → biasanya membaik setelah defekasi
- Perubahan **frekuensi** BAB
- Perubahan **konsistensi** BAB

IBD → Disease → ada kelainan struktural
IBS → Syndrome → tidak ada kelainan struktural
