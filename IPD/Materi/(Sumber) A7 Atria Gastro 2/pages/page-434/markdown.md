# Skistosomiasis

## Patofisiologi:

- Siklus hidup:
1. Telur menetas → mirasidia
2. Infeksi siput → serkaria
3. Serkaria menembus kulit manusia
4. Masuk sirkulasi → paru → jantung → hepar (berkembang)
5. Setelah dewasa keluar lewat vena porta → ke vena mesenterika (S. mansoni &amp; japonicum)/pleksus vena pelvis (S. haematobium)
