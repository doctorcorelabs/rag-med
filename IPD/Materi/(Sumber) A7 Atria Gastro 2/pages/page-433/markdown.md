# Skistosomiasis

Definisi: Infestasi akibat cacing genus Schistosoma

Sinonim: Bilharziasis, Snail fever

## Etiologi:

- Schistosoma mansoni → Afrika, Amerika Selatan, Karibia
- Schistosoma haematobium → Afrika, Timur Tengah
- Schistosoma japonicum → Cina, Asia Tenggara

Daerah endemis Schistosoma di Indonesia: Napu, Bada, Lindu; Sulawesi Tengah
