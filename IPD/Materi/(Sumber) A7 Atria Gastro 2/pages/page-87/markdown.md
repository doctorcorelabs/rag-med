# Hernia Inguinalis

## Tata Laksana

- Pembedahan → tata laksana definitif
- Herniotomy
- Pengangkatan kantong hernia (dilakukan pada kasus anak)
- Herniorrhapy
- Herniotomy + menguatkan dinding posterior kanalis inguinalis dengan penjahitan
- Teknik: Shouldice, Bassini, McVay
- Hernioplasty
- Herniotomy + menguatkan dinding posterior kanalis inguinalis dengan mesh
- Teknik: Lichtenstein, plug and patch
