# Ascariasis

## Manifestasi Klinis:

- Umumnya asimtomatis
- Gejala awal
- **Sindrom Loeffler**: akibat migrasi larva ke paru-paru
- batuk kering, mengi, dan eosinofilia paru sementara
- Gejala lanjut (intestinal)
- Anoreksia, nyeri perut, mual muntah, diare
- Obstruksi usus → apendisitis, pankreatitis, perforasi usus
