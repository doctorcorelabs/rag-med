# Penyakit Crohn

Derajat Keparahan berdasarkan Manifestasi Klinis

- Dinilai menggunakan Crohn Disease Activity Index (CDAI), dengan pembagian:

|  Derajat | Manifestasi  |
| --- | --- |
|  Remisi klinis | Asimtomatis, baik spontan maupun setelah pengobatan
Bila remisi akibat kortikosteroid maka disebut “dependen steroid”  |
|  Ringan (mild) | Gejala ringan, penurunan berat badan <10%, tidak ada gejala sistemik, dan tidak ada gejala obstruksi saluran cerna  |
|  Sedang-berat (moderate-severe) | Gagal pengobatan derajat ringan dan terdapat gejala sistemik (demam, nyeri abdomen, anemia) atau gejala obstruksi saluran cerna  |
|  Sangat berat (severe-fulminant) | Gejala sistemik dan obstruksi saluran cerna menetap meskipun telah diberikan kortikosteroid dan immunomodulator  |
