# Disentri Basiler

## Patofisiologi

- Tertelan → replikasi di usus halus → kolon
- Invasi langsung epitel usus → menyebar dari sel ke sel → kematian sel dan pengelupasan mukosa → inflamasi usus → gangguan absorpsi → diare
- Produksi toksin (shiga toxin)
- Lesi vaskular kolon → diare berdarah
- Lesi vaskular ginjal → *Hemolytic uremic syndrome* (HUS)
- Trombositopenia, gagal ginjal akut, anemia hemolitik
