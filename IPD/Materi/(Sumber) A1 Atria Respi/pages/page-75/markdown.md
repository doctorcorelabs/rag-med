# Bronkitis Akut

## Pemeriksaan Penunjang:

- Bronkitis akut cukup ditegakkan dengan anamnesis dan pemeriksaan fisik
- Rontgen thoraks → untuk mengevaluasi penyakit lain, seperti pneumonia, PPOK
- Pemeriksaan mikrobiologi → hanya bila ada indikasi khusus, misalnya pada kecurigaan infeksi bakteri atau pada pandemic COVID-19
