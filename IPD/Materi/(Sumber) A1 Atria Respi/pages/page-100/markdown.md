# Bronkiolitis

## Tata Laksana

- Bronkiolitis tidak berat
- Penyakit akan membaik minggu pertama dan swasirna dalam 2-3 minggu.
- Edukasi tanda bahaya seperti apnea, sianosis, tidak mau makan/minum, demam tinggi, dan sesak napas untuk segera datang ke fasilitas kesehatan terdekat.
- Terapi simtomatis
