# Pneumonitis Hipersensitif

## Manifestasi Klinis

- Fase akut (4-8 jam sejak paparan): demam, lemas, lelah, sesak napas, sakit kepala, ruam tubuh
- Fase subakut (harian-mingguan): batuk produktif, anoreksia, sesak napas lebih berat dibanding kondisi akut
- Fase kronis: gejala tidak prominen, namun telah terjadi proses kronik intraparu → fibrosis (bird’s lung), emfisema (farmer’s lung), destruksi alveolar, clubbing finger

Fase akut disebabkan oleh kompleks imun (Hipersensitivitas tipe III)
Sedangkan fase subakut dan kronis diperantarai oleh aktivitas sitotoksik T-helper (Hipersensitivitas tipe IV)
