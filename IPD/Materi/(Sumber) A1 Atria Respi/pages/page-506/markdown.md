# Definisi Kasus SARS

- Suspek → Demam (&gt;38°C) + Batuk/Sesak + kontak erat/perjalanan/tinggal di daerah dengan transmisi lokal SARS
- Probabel → Kasus suspek + Gambaran radiologi konsisten dengan pneumonia/ARDS atau dengan pemeriksaan PCR/serologi/virologi positif untuk SARS
- TIDAK ADA KASUS KONFIRMASI pada SARS
