# Croup

## Manifestasi Klinis

- Gejala prodromal → demam subfebris 12-72 jam, disertai hidung berair (rinore), nyeri menelan dan batuk ringan
- Batuk mengonggong (barking cough)
- Suara menjadi serak
- Stridor inspiratori
- Retraksi dinding dada, takipnea, takikardi
- Perburukan bila terjadi infeksi sekunder bakterial → trakeitis bakterial
