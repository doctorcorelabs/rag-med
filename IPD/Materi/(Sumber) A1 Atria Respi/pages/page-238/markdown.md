# Tuberkulosis

Hasil Pengobatan

|  Hasil | Definisi (Disederhanakan)  |
| --- | --- |
|  Sembuh | Awal pengobatan: bakteriologis positif
Akhir pengobatan: BTA sputum atau biakan negatif dan hasil pemeriksaan negatif pada salah satu pemeriksaan sebelumnya  |
|  Pengobatan Lengkap | Pengobatan lengkap namun tidak memiliki hasil BTA atau biakan negatif pada akhir pengobatan karena tidak dilakukan atau hasilnya tidak ada  |
|  Pengobatan Gagal | Bulan kelima atau akhir pengobatan: BTA sputum atau biakan positif  |
|  Putus Obat | Tidak memulai pengobatan setelah terdiagnosis TB atau menghentikan pengobatan selama 2 bulan berturut-turut atau lebih  |
|  Meninggal | Pasien TB yang meninggal dengan alasan apapun sebelum atau selama pengobatan TB  |
