# Case Description

Bayi Ny. Pink, usia 1 bulan dirawat di NICU karena lahir prematur. Pasien lahir pada usia kehamilan 31 minggu. Pasien hingga saat ini masih mendapatkan terapi oksigen karena mengalami desaturasi bila selang oksigen dilepas. Hari ini kondisi pasien mengalami perburukan, didapatkan keadaan umum sesak, penurunan berat badan, RR 70 kali/menit, suhu afebris. Pada PF tampak napas cuping hidung (+), retraksi intercostal (+).
