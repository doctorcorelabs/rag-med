# Pneumotoraks

Tension pneumothorax → diagnosis klinis

## KLINIS PNEUMOTHORAKS

1. Distres napas
2. Ekspansi dinding dada tertinggal
3. Fremitus taktil menurun
4. Perkusi hipersonor
5. Suara napas menurun atau menghilang

## INSTABILITAS HEMODINAMIK

1. Hipotensi
2. Takikardia
3. Sianosis
4. Penurunan kesadaran
5. Peningkatan JVP

## DEVIASI MEDIASTINUM

1. Deviasi trakea kontralateral
2. Deviasi apeks jantung
