# Pneumonia

## Etiologi:

- CAP tipikal → Streptococcus pneumoniae (rust colored sputum), Haemophilus influenzae, Staphylococcus aureus, dan basil Gram-negatif (Klebsiella pneumoniae dan Pseudomonas aeruginosa)
- CAP atipikal → Mycoplasma pneumoniae, Chlamydia pneumoniae, Legionella sp., dan respiratory viruses
- HAP → anaerob
- VAP → Pseudomonas aeruginosa, Acinetobacter, MRSA
