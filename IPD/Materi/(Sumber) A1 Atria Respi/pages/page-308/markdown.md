# Emboli Paru

## Pemeriksaan Penunjang:

- CT Angiografi Paru (CTPA) → pemeriksaan diagnostik definitif
- Melihat arteri pulmonalis dengan kontras
- Saddle embolism → bila terlihat thrombus di bifurkasi arteri pulmonalis
- Sensitivitas dan NPV tinggi → bila D-dimer test rendah, sangat tidak mungkin seseorang menderita emboli paru
- Spesifisitas rendah → D-dimer tinggi tidak berarti pasien menderita emboli paru
- Kontraindikasi: kehamilan, gangguan ginjal
