# Tata Laksana Controller Asma Dewasa

|  Langkah 1
(Asma intermiten) | gejala < 2 kali/bulan | ICS dosis rendah - formoterol (bila diperlukan)  |
| --- | --- | --- |
|  Langkah 2
(Asma intermiten) | gejala >2 kali/bulan ATAU terbangun malam hari <1 kali/minggu  |   |
|  Langkah 3
(Asma persisten ringan) | gejala hampir setiap hari atau terbangun malam hari karena asma >1 kali/minggu | ICS dosis rendah + LABA  |
|  Langkah 4
(Asma persisten sedang) | Kriteria Langkah 3 + Penurunan fungsi paru | ICS dosis menengah + LABA  |
|  Langkah 5
(Asma persisten berat) | Bila respon pengobatan Langkah 4 tidak adekuat | ICS dosis tinggi + LABA  |

Bila langkah sebelumnya tidak adekuat, maka naik ke langkah berikutnya.
