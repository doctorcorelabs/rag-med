# Altitude Illness

## Manifestasi Klinis

|  Tipe | Manifestasi klinis  |
| --- | --- |
|  AMS | • Onset dalam beberapa jam dan hilang setelah 1-3 hari
• Nyeri kepala, mual, muntah, anoreksia, lemas, gangguan tidur, pusing  |
|  HACE | • Ensefalopati → penurunan kesadaran, ataxia, defisit neurologis fokal  |
|  HAPE | • Batuk (berdarah, pink frothy sputum), sesak
• Ronkhi, takikardi, takipnea  |
