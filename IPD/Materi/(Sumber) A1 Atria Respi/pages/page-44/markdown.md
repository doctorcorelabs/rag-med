# Massa Plica Vocalis

- Nodul → bilateral pada sisi medial plika vokalis bagian 1/3 anterior bewarna putih keabuan
- Polip → massa bertangkai
- Granuloma → unilateral/bilateral pada glotis posterior dekat dengan prosesus vokalis kartilago aritenoid, ukurannya lebih besar dari nodul
- Papilloma → massa seperti bunga kol warna kelabu/kemerahan
- Karsinoma → massa yang mudah berdarah
