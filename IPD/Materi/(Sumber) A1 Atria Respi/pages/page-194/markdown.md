# Pneumonia Aspirasi

Definisi: Inflamasi parenkim paru akibat inhalasi benda asing ke dalam saluran napas. Bila substansi bersifat steril, maka disebut pneumonitis aspirasi

## Faktor Risiko:

- Kesadaran menurun → alkohol, sedasi, anestesi, stroke
- Penyakit neurodegenerative → Parkinson, Alzheimer
- GERD, gangguan motilitas esofagus
- Kongenital: Tracheoesophageal fistula
