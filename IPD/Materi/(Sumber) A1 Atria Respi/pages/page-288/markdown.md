# Efusi Pleura

## Manifestasi Klinis:

- Asimtomatis bila efusi &lt;300 mL
- Sesak napas, nyeri dada pleuritik, batuk kering
- Inspeksi → hemithoraks ipsilateral tertinggal saat inspirasi
- Perkusi → efusi menimbulkan suara **redup**, dengan batas redup meninggi pada dinding dada lateral (garis Ellis Damoiseau)
- Auskultasi → **pleural friction rub (+)**, suara napas menurun.
