# Emboli Paru

## Tata Laksana

### Stabilisasi ABC
- Oksigenasi → bila SaO₂ 90%
- Norepinefrin 0,2-1 mikrog/kg/min → dilakukan pada kasus syok kardiogenik

### Antikoagulan → pada pasien high/moderate-risk
- LMWH (Enoxaparin) atau Fondaparinux → first line
- Unfractionated Heparin → pilihan kedua bila LMWH tidak tersedia
- Setelah 5 hari, tambahkan Warfarin → **gold standard**

Warfarin membutuhkan waktu ~5 hari hingga betul-betul efektif, sehingga dibutuhkan terapi heparin terlebih dahulu.
