# Bronkiektasis

## Tata Laksana

- Pencegahan eksaserbasi dapat dilakukan dengan:
- Bronchopulmonary hygiene → fisioterapi dada regular, nebulisasi salin hipertonis
- Antibiotik sebagai pencegahan → bila terjadi &gt;2 kali rekurensi / tahun
- Azithromycin 1 x 250 mg jangka panjang
- Mukolitik
- Invasif → lobektomi, transplantasi paru
