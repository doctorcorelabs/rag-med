# Epiglotitis

## Pemeriksaan Penunjang

- Rontgen leher lateral → tampak pembesaran epiglotis (thumbprint sign)
- Hematologi rutin → leukositosis
- Kultur darah → identifikasi patogen
- Laringoskopi → konfirmasi diagnosis, dilakukan bila
- Tidak ada obstruksi jalan nafas
- Foto rontgen tidak khas untuk epiglotitis
