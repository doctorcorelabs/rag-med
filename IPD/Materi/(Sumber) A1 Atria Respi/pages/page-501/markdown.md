# COVID-19

- Infeksi saluran napas akibat SARS-CoV-2
- Definisi kasus:
- Suspek
- Demam + batuk dengan riwayat tinggal/kerja di tempat berisiko tinggi selama 14 hari
- Asimptomatis namun antigen COVID-19 (+)
- Probabel → Kriteria klinis DAN kontak erat dengan kasus probable/konfirmasi
- Konfirmasi → RT-PCR (+) atau kasus suspek/probable dengan antigen (+)
- Tata laksana: isolasi mandiri, antivirus, vitamin C, vitamin D
