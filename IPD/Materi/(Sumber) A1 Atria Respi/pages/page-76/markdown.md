# Bronkitis Akut

## Tata Laksana:

Bronkitis akut biasanya bersifat swasirna, pengobatan untuk simtomatis saja

- Dextrometorphan, Ambroksol, Guaifenesin, Kodein*
- Asetaminofen, NSAID → antipiretik, analgesik
- Salbutamol → bila wheezing cukup parah
- Antibiotik → sangat jarang digunakan, kecuali bila terbukti ada infeksi bacterial
- Kondisi penyerta (seringkali common cold): antihistamin, pseudoefedrin

*Kodein harus digunakan secara hati-hati karena potensi adiktif dan efek sampingnya. Pada beberapa guideline sudah tidak dianjurkan penggunaan kodein sebagai antitusif
