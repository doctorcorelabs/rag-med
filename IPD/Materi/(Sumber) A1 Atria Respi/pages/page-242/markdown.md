# Pemantauan Respon Pengobatan

Pemantauan pada pasien yang menerima OAT lini pertama kategori 1 (2 RHZE/4 RH)

- Akhir fase intensif (akhir bulan ke-2)
- Negatif → lanjutkan pengobatan
- Positif → lanjutkan pengobatan, periksa ulang pada akhir bulan ke-3

- Akhir bulan ke-3 (hanya bila BTA positif pada akhir fase intensif)
- Negatif → lanjutkan pengobatan
- Positif → pemeriksaan kepekaan obat dengan TCM atau biakan
