# Flu Burung

Rangkuman

|  Klasifikasi Kasus | Kriteria Sederhana | Tata Laksana  |
| --- | --- | --- |
|  Kelompok Risiko Tinggi | Tanpa gejala klinis namun kontak erat dengan unggas atau pasien suspek H5N1 | Profilaksis: Oseltamivir 1x75 mg/hari (maks 6-8 minggu)  |
|  Suspek | Kontak erat dengan gejala klinis | Pengobatan: Oseltamivir 2x75 mg/hari (5 hari) Zanamivir 2x10 mg/hari (5 hari)  |
|  Probabel | Suspek + PP non-spesifik (mis: kenaikan titer antibody H5)  |   |
|  Konfirmasi | Suspek/probabel + PP spesifik (mis: PCR, isolasi virus)  |   |

Sumber: PPK PAPDI, Pedoman Tata Laksana Flu Burung (2010), Pedoman Penanggulangan Flu Burung (2017)
