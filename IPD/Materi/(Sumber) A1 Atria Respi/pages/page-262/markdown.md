# Pneumocystis Pneumonia

## Pemeriksaan Penunjang

### Darah
- Pemeriksaan CD4 → biasanya di bawah 200 sel/mikroL
- Peningkatan LDH
- Pemeriksaan 1-3 beta D-glucan → komponen dinding sel *P. jirovecii*

### Radiografi
- Rontgen paru → infiltrate difus bilateral dan interstisial
- CT scan → dapat digunakan pada kecurigaan PCP dengan gambaran rontgen normal
