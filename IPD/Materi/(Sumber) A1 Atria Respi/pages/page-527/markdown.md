# Case Description

Tn. Wreck, usia 34 tahun dilarikan ke RS di kaki gunung karena sesak dan batuk berdahak warna merah muda. Pasien sedang mendaki gunung dengan teman-temannya hingga ketinggian 2.500 mdpl dan sempat mengeluhkan pusing dan mual. Pada pemeriksaan didapatkan TD 120/80 mmHg, HR 120 kali/menit, RR 32 kali/menit, suhu afebris. Pada PF didapatkan ronkhi di kedua lapang paru.
