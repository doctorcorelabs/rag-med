# Acute Respiratory Distress Syndrome

## Manifestasi Klinis:

- Tampak sakit berat + multi organ failure
- Sesak yang cepat memburuk → butuh oksigen konsentrasi tinggi hingga alat bantu nafas
- Pemeriksaan fisik:
- Takipnea, takikardia, hipo/hiper-termi
- SaO₂ rendah → butuh bantuan oksigen konsentrasi tinggi
- Akral dingin, sianosis, CRT melambat → sepsis
- Pada paru, dapat ditemukan ronki bilateral seluruh lapang paru
