# COVID-19: Manifestasi Klinis

## ASIMPTOMATIS

### Kriteria Diagnosis
- Tidak memiliki gejala apapun
- Hanya terdeteksi dengan pemeriksaan penunjang

## RINGAN

### Kriteria Diagnosis
- Tidak ada bukti pneumonia DAN hipoksia
- Gejala:
- Demam
- Batuk
- Lemas
- Anoreksia
- Napas pendek
- Mialgia
- Nyeri tenggorokan
- Kongesti hidung
- Sakit kepala
- Diare
- Mual-mutah
- Anosmia: hilang penghidu
- Ageusia: hilang pengecap
