# Efusi Pleura

## Diagnosis:

- Setelah konfirmasi efusi pleura → **BEDAKAN** transudatif/eksudatif
- **Torakosentesis →** analisa cairan pleura
- Kualitatif: tes Rivalta → mengendapkan protein pada suasana pH 4-5 (aquadest + asam asetat + sampel)
- Positif: keruh, terbentuk endapan → eksudat
- Negatif: tetap jernih → transudat
