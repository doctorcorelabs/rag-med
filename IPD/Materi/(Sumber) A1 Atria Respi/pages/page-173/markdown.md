# Pneumonia

## Indikasi Rawat → berdasarkan skor CURB-65

- Skor 0: Rawat jalan
- Skor 1: Rawat jalan atau rawat inap
- Bila skor 1 akibat usia saja: rawat jalan
- Bila skor 1 akibat lainnya: rawat inap non-ICU
- Skor 2: Rawat inap non-ICU
- Skor ≥3: Rawat inap ICU*

|  Skoring CURB-65 (masing-masing 1 poin)  |   |
| --- | --- |
|  C | Confusion (disorientasi, penurunan kesadaran)  |
|  U | Urea (BUN) > 20 mg/dl  |
|  R | Respiratory rate ≥ 30 kali/menit  |
|  B | Blood pressure (TD) sistolik ≤ 90 mmHg atau diastolic ≤ 60 mmHg  |
|  65 | Usia ≥ 65 tahun  |

Number:

CURB-65: Pneumonia PDPI

Interpretasi CURB-65: Harrison 19th ed.
