# Pneumokoniosis

Definisi: Penyakit paru interstisial yang disebabkan karena akumulasi debu mineral atau logam serta reaksi jaringan terhadap partikel tersebut

Etiologi → inhalasi partikel iritan
- Asbestosis → serat asbestos
- Silicosis → debu silika
- Coal workers' pneumoconiosis (CWP) → debu batu bara
- Beriliosis → logam berilium
- Byssinosis → debu kapas
- Aluminosis → debu aluminium
