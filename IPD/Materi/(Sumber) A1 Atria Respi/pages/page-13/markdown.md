# Pertusis

## Pemeriksaan Penunjang

- Kultur dari swab nasofaring → diagnosis pasti
- Darah lengkap → leukositosis (limfositosis absolut)
- X-ray toraks → infiltrat perihiler, atelektasis, epiema
- Serologi → IgG anti-toksin Pertusis
