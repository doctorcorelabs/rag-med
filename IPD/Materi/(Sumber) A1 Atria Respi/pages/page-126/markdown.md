# PPOK

## Pemeriksaan Penunjang:

- Spirometri → **dasar diagnosis**, **kriteria FEV₁ / FVC &lt;70%**, **tidak boleh dilakukan saat serangan**
- Laboratorium darah: Hb, Ht, Leukosit
- AGD → untuk menilai gagal nafas
- Rontgen thoraks:
- Emfisema: hiperinflasi, hiperlusen, diafragma mendatar, jantung pendulum
- Bronkitis kronis: corakan bronkovaskular bertambah
