# Bronkiolitis

## Pemeriksaan Penunjang

- Saturasi oksigen:
- ≤92% → perawatan intensif
- &gt;94% → rawat jalan
- Foto toraks → tidak perlu dilakukan bila gejala tipikal bronkiolitis
- Temuan tidak spesifik, berupa hiperinflasi, penebalan peribronkial (peribronchial cuffing), dan atelektasis
